Tower Defense
TOWER_DEFENSE.EXE — Jeu de stratégie en Java

Description

Jeu de Tower Defense développé en Java. Le joueur place des tours défensives sur une carte pour empêcher des vagues d'ennemis d'atteindre la base.

Gestion des vagues, des types d'ennemis, des différentes tours avec portée et dégâts variables, et d'un système de score.

Technologies

Java JavaFx
